Sameer's File Metadata Microservice

User Stories

  • I can choose a file from my computer and upload it on to the server.
  • When I upload the file, I get a JSON response with the file's metadata information.

Example Output

{ Name: "planet.ico", Type: "image/vnd.microsoft.icon", Size: "117 kilobytes" }
